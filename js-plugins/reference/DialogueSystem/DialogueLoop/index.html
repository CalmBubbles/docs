DialogueLoop

class / Implemented in DialogueSystem

Description

The class that updates dialogue tasks and time.

Static Properties

targetFrameRate The refresh rate at which the system tries to update.
timeScale The scale which time passes.
maximumDeltaTime The maximum value of DialogueLoop.deltaTime. This also limits the increase in DialogueLoop.time between two frames.
frameCount (Read Only) The number of frames since initialization.
unscaledTime (Read Only) The timeScale-independent time in seconds since initialization.
unscaledDeltaTime (Read Only) The timeScale-independent interval in seconds between the last and current frame.
time (Read Only) The time in seconds since initialization.
deltaTime (Read Only) The interval in seconds between the last and current frame.

Static Methods

Append Adds a method/task to be updated.

Static Asynchronous Methods

Delay Waits for a specific amount of time.
