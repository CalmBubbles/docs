DialogueStyle

class / Implemented in DialogueSystem

Description

Class for the styles of characters.

This is used in the same way as the style property of a HTMLElement.

Most documentation here would just be from MDN Web Docs.

Constructor

DialogueStyle Creates a new character style.

Static Properties

identity (Read Only) The default style of a DialogueManager.
