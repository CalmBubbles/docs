DialogueTextAnimation

class / Inherits NamedItem / Implemented in DialogueSystem

Description

Base behavior class for text animations.

Inherit this class when making text animations.

Properties

loop Tells if the animation should loop.
wait Should the system wait for this animation to finish before moving on to the next one.
duration The animation duration in seconds.
hasEnded (Read Only) Tells if the animation has ended.
index (Read Only) The index of the character this animation is attached to.
time (Read Only) The time that this animation has passed in seconds.
loopCount (Read Only) Amount of loops this animation has taken.
dialogueChar (Read Only) The character this animation is assigned to.

Methods

Animate Animates a frame of this animation.

Overridable Methods

OnSet Called once the animation is set.
Update Called every frame for animations.
OnEnd Called when the animation has ended.

Inherited Properties

name Name of the item.
