DialogueManager.ClearText

async void ClearText ();

Description

Clears the dialogue line.

Use this if you only want to clear text. 

JavaScriptawait diaManager.Type("The only non themed portrait I have here is", 1);
await diaManager.Type("...", 0.04);
    
// Clear text
await diaManager.ClearText();
    
// Use portrait
await diaManager.Type("This horrendous thing.", 1, new DialogueData({ portrait : "sus" }));
await DialogueLoop.Delay(1.6);
    
// This will not clear the portrait
await diaManager.ClearText();
