DialogueEvent

enumeration / Implemented in DialogueSystem

Description

The events called during manager tasks.

Properties

None (Default) No event.
Load Called after all resources have loaded.
Enable Called when the manager gets enabled.
Disable Called when the manager gets disabled.
TypingEnter Called before text shows when typing.
TypingExit Called after the text has shown when typing.
PortraitEnable Called when the dialogue portrait enables.
LineClear Called when the dialogue line gets cleared.
Clear Called when the dialogue box gets cleared.
