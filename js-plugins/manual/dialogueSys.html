First Few Steps

Let's start by adding a div

HTML: 

<div></div>
Let's see what does that look like...
What am I looking at?


Let's add style so we can actually see something. We will also add the font we are going to use.

CSS: 

body
{
    background : black;
    color : white;
    font-family : lightweight;
    font-size : 20px;
    line-height : 30px;
}

/* I'm going to name my div's class "text-box" */
.text-box
{
    background : black;
    max-width : 320px;
    min-height : 90px;
    max-height : 90px;
    padding : 10px;
    border : 4px solid white;
    margin : 5px auto;
    overflow-wrap : break-word;
}

@font-face
{
    font-family : "lightweight";
    src : url(/fonts/lightweight.ttf);
}

HTML: 

<!-- add the class you made -->
<div class="text-box"></div>
That's better


Now let's ready our css animations and dialogue box disabled state

CSS: 

/* I'm going to name my 'in' animation "textBoxIn" */
@keyframes textBoxIn
{
    from { transform : scale(1, 0); }
    
    to { transform : none; }
}

/* I'm going to name my 'out' animation "textBoxOut" */
@keyframes textBoxIn
{
    from { transform : none; }
    
    to { transform : translate(0, -100%); }
}


/*
When the dialogue box is disabled these values
will be given to the div

The attribute 'data-disabled' tells us if the
dialogue box is diabled
*/
.text-box[data-disabled="true"]
{
    transform : scale(1, 0);
}
If we run the code, we will see nothing has changed. This is because we haven't made a dialogueSys object yet.

Making a Dialogue System Object

We can make a dialogue system object by assigning a variable with a dialogueSys class.

JS: 

// you can get the box element by using querySelector
let dialogueBox = document.querySelector("#dialogue-box");

// make the dialogueSys object
var diaSys = new dialogueSys(
    // the box element
    dialogueBox,
    
    // letters or string chars that will not make
    // a sound when added to the box
    //
    // I've set this null for now
    null,
    
    // people or characters that will show in
    // an image portrait
    //
    // I've set this null for now
    null,
    
    // animation 'in'
    "textBoxIn linear",
    
    // speed of animation 'in' in seconds
    0.125,
    
    // animation 'out'
    "textBoxOut",
    
    // speed of animation 'out' in seconds
    0.125,
    
    // where the image portrait is located
    "res/img",
    
    // where the dialogue audio is located
    "res/audio"
);
If we run that, we will see that the box disappeared.
