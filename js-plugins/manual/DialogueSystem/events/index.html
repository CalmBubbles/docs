Events

Events are used to figure out the task state of a manager. Technically, events are events.

There are 8 events in the manager. They happen in different times and circumstances.
They can be found at the DialogueEvent enumeration, but here they are:

Load - when the manager and it's resources has loaded
Enable - when the manager enables
Disable - when the manager disables
TypingEnter - when the typing iteration is about to started
TypingExit - when the typing iteration has ended
PortraitEnable - when the portrait enables
LineClear - when the text line clears
Clear - when the manager clears entirely
