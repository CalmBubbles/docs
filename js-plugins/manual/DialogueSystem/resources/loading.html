Loading resources

Loading resources is easy. For sprites, just create a new DialogueSprite object. 

JavaScript
const sprite = new DialogueSprite(
    // Name for identification
    "",
    
    // Source file
    ""
);
And for audio clips, just do the same from sprites. But use DialogueAudioClip instead. 
JavaScript
const audio = new DialogueAudioClip("", "");
And to find out if they're loaded and ready for use, just check their isLoaded property. 
JavaScript
alert(`loaded sprite: ${sprite.isLoaded}\nloaded audio: ${audio.isLoaded}`);
