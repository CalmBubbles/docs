Clearing everything

To clear everything in the dialogue box, you can use the manager's asynchronous Clear method.

This doesn't actually clear everything. The only thing that gets cleared are elements with the class .dialogue-line-level. This includes all of the text, spaces and line breaks. Then the dialogue portrait only gets disabled and hidden, not cleared.

No example here, this just works similar to the ClearText method.
