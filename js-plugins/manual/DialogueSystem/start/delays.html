Adding delays

To add delays between Type calls, you can use a DialogueData and its delayBefore and delayAfter properties. delayBefore adds a wait before the typing iteration, and delayAfter adds a wait after the iteration. The delay properties are measured by seconds. 

JavaScriptawait diaManager.Type("You can now add pauses, ", 1);
await diaManager.Type("like this.", 1, new DialogueData({
    delayBefore: 0.5,
    delayAfter: 2
}));
await diaManager.ClearText();
await diaManager.Type("The delay also affects the entire sequence.", 1, new DialogueData({
    delayAfter: 2.5
}));

// gets called after the delay
alert("RAAAAHHHHHHHHHHHHH!!!!!");

The only downside in using the Type method to add delays, is that delays do not work when the manager is disabled. To add delays that do not get affected by the manager's state, head here.
