Speeding up tasks

Welcome to the end of the topic! To finish things off, we're gonna finish things off.

To speed up a task, the manager has a Finish method. This method skips every time delay and finishes to the end of the task execution. It does not break the execution order, it only speeds it up to an instant. This only works on the asynchronous methods of the manager.

Once a sped up task finishes, the effect of the Finish method ends. The next task will not be sped up to an instant, and will require the Finish method to be called again to do so.

If placed in the sequence, it must be before the target task that will be sped up. 

JavaScriptdiaManager.Finish();
await diaManager.Type("This text is shown in an instant.\n", 1);
await diaManager.Type("But this one is not.", 1, new DialogueData({ delayBefore: 1 }));
