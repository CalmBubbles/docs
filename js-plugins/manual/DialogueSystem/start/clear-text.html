Clearing text

To clear text, simply call the manager's ClearText asynchronous method. 

JavaScriptawait diaManager.Type("... ", 0.01);
await diaManager.Type("huh", 0.8);
await diaManager.Type("...", 0.01);
await diaManager.ClearText();             // here
await diaManager.Type("WHAT THE F-", 2);
await diaManager.ClearText();             // also here
