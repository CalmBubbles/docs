Setting up

The first thing to do after installation, is to set up the project. But of course, you probably already have it set, like base styles and such.

For this getting started, we'll start from square one. You don't have to though, just try to follow along and understand.


Structure

DialogueSystem.js uses HTML, which is composed of elements. You need to know what elements are going to be used by the system.

To start simple, lets focus on the two major elements, the dialogue box and the dialogue line.


The dialogue box is the very main element used by the system. Its the element wrapping up every entire element used, including the dialogue line. Used for moving everything around and for making the box around dialogues, hence the name dialogue "box".

The dialogue line is the element whose purpose is entirely housing all the text displayed. This is where all the text characters pop out of.


For the system to work you must have one div element to serve as the dialogue box, strictly a div element. I know it looks blank, but trust me that it will look better soon. You don't have to add in the other elements, as they will be added automatically. 

HTML<!-- You can just add a normal div element -->
<div></div>


Style

To start styling the two elements, you can use their CSS class selectors. .dialogue-box for the dialogue box, .dialogue-line for the dialogue line. You don't have to add these classes to the elements, this is auto.

Remember that the dialogue box is required to have a fixed width and at least a minimum height to contain all of its elements. And in order for text wrapping to work correctly, it is required for the dialogue line to have a fixed width and for its font size and spacing to be proportional to the width.

For this manual, we'll be using these styles. 

CSS.dialogue-box
{
    background : black;

    /* It may not look like it, but dynamic units like `vw` and `%` can
       work like fixed units if everything is proportional to each other */
    width : 91.67vw;
    height : 18.06vw;
    padding : 2.78vw;
    border : 0.83vw solid white;
    font-family : "lightweight"; /* Use any font */
    font-size : 4.44vw;
    line-height : 6.11vw;
    margin: auto;
}

.dialogue-line
{
    width: 100%;
}

When you apply these styles, they do not get displayed yet. But you can preview them by making a mockup of the dialogue box structure. Of course, you have to remove that mockup after previewing it. 

HTML<!--
    Never add any whitespaces or line breaks in the mockup.
    The dialogue line is also required to be a div
-->
<div class="dialogue-box"><div class="dialogue-line">This is a mockup preview.</div></div>


Script

To get started on scripting, you have to get the script installed. Then you have to create a DialogueManager object and pass in the dialogue box element as a parameter. 

HTML<!-- I added an id to avoid using `document.querySelector`, because I want to >:)) -->
<div id="textbox"></div>
JavaScriptconst diaManager = new DialogueManager(
    textbox // the dialogue box
);

Now if we open the page, we can actually see the dialogue box.

This is no longer a mockup preview. You can now move on to the next topic.
