The Type method

This is probably the method you've been waiting for. The method that pops up text like in RPGs.

To type onto the manager, you can use its asynchronous Type method. You pass in the text string as the first parameter, and the typing speed on the second. Both parameters are required. You also need need to have the manager enabled for this. 

JavaScriptdiaManager.Once(DialogueEvent.Load, async () => {
    await diaManager.SetActive(true); // required to be awaited

    await diaManager.Type(
        "Reeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeee", // text
        1 // speed
    );
});

For the typing speed, when higher the value, the faster text iterates. The wait interval between between each text character is calculated by 1 / 60 / speed milliseconds. It's recommended normal value is 1, where the text characters have an interval of 16.67ms.
