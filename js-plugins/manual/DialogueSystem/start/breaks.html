Adding line breaks

There are two ways of adding text line breaks. By using the newline escape character ("\n") within the Type method's text parameter. And by using the manager's BreakLine method.

You can use any, it doesn't really matter that much. You can even use both.


Within the Type method

In the Type method, you can add a newline escape character ("\n") on the part you want to start a line break. 

JavaScriptawait diaManager.Type("Before break\nAfter break", 1);

This is a simple way of breaking a line. The only con with this, is that the "\n" is considered as a text character. Like other characters, they have a wait interval before popping out. If the typing speed is slow, the line breaks will too. And the more "\n" you add, the longer the delay between those lines.


BreakLine method

Another way of breaking lines is by using the manager's BreakLine method. Unlike using an escape character, this is immediate, for the cost of having Type calls separated. 

JavaScript// sandwiches for this break
await diaManager.Type("Before break", 1);
diaManager.BreakLine();
await diaManager.Type("After break", 1);

You can also change the amount of breaks by passing in the target amount as a parameter. And of course, by default its 1.
