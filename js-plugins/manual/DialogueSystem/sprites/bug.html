Broken text line

With portraits enabled, it may break the dialogue line. To fix this, there is a data-portrait-enabled HTML attribute within the dialogue box. It's value can either be true or false.

It can be used within CSS to fix the text line problem, by creating a new style to go on top the default style of the dialogue line. It can be used like: 

CSS.dialogue-box[data-portrait-enabled="true"] .dialogue-line
{
    /* ... your style fix here */

    /* The one used in example: */
    width: calc(var(--scaler) * 888vw);
}
