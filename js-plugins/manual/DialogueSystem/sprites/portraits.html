Portraits

One of the things usually seen in dialogue systems, are portraits. They are used to show characters and their emotion. They are easy to do, and they have a lot of impact.

To use sprites as portraits, add the sprites in the sprite array of your manager config. This will make the sprites accessible by the manager. 

JavaScriptdiaManager.config.sprites = new DialogueSpriteArray(
    [
        { name: "niko", src: "niko.png" },
        { name: "niko4", src: "niko4.png" },
        { name: "niko3", src: "niko3.png" },
        { name: "niko_pancakes", src: "niko_pancakes.png" },
        { name: "niko2", src: "niko2.png" },
        { name: "niko_what", src: "niko_what.png" },
        { name: "niko_eyeclosed2", src: "niko_eyeclosed2.png" },
        { name: "niko_83c", src: "niko_83c.png" },
        { name: "niko_speak", src: "niko_speak.png" },
        { name: "niko_smile", src: "niko_smile.png" },
        { name: "niko5", src: "niko5.png" },
        { name: "niko6", src: "niko6.png" },
        { name: "magpie", src: "magpie.png" },
        { name: "magpie_smile", src: "magpie_smile.png" },
        { name: "magpie_hm", src: "magpie_hm.png" },

// For just `DialogueSprite` objects
// you can just push them to the array
// like this:
//
// diaManager.config.sprites.items.push(
//     new DialogueSprite("niko", "niko.png")

Then call the Type method with a DialogueData that has its portrait property set to the name of the sprite you want to use. 

JavaScriptawait diaManager.Type("Is that alcohol?", 1, new DialogueData({
    audio: "default",

    // Use portrait with name "niko"
    portrait: "niko"
}));

Now you can style the portrait using CSS by using the .dialogue-portrait selector. It is recommended for the dialogue box to have a minimum height that fits the portrait. 

CSS.dialogue-portrait
{
    /* ... your styles here */

    /* The one used in example: */
    float: right;
    margin-top: calc(var(--scaler) * 35.0789vw);
    margin-right: calc(var(--scaler) * 35.0789vw);
    width: calc(var(--scaler) * 208.939vw);
}
