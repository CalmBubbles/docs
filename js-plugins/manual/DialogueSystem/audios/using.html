Using audios

Audios can reduce boringness of dialogues, especially the ones without portraits. It's possible by making sound play for every text character typed.

To do that, you gotta add your audio clips in the audio array of your manager config. This makes those clips accessible by the manager 

JavaScript
Now you can call the Type method with a DialogueData that has its audio property set to the name of the clip you want to hear. 
JavaScript
To change how frequent audio plays, use the manager's soundInterval property.
