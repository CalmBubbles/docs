Making a mix

You can mix up DialogueStyle objects into one.

First you need to have a DialogueStyle object, which will be the default style of the mix. Then you can call the default style's Create method and pass the other styles onto it. The Create method will return a new DialogueStyle object, containing the mixed properties. 

JavaScript// default style
const defaultStyle = new DialogueStyle({
    color: "lightblue",
    fontSize: "1.2em",
    textDecoration: "underline",
    fontFamily: "monospace",
    fontWeight: "bold"
});


// other styles
const whiteText = new DialogueStyle({
    color: "white"
});

const smolSize = new DialogueStyle({
    fontSize: "0.1em"
})

const noBolderline = new DialogueStyle({
    fontWeight: "normal",
    textDecoration: "none"
});


// new style
const newStyle = defaultStyle.Create(
    whiteText,
    noBolderline,
    new DialogueStyle({
        fontWeight: "200"
    }),
    smolSize
);

// this is also just
new DialogueStyle({
    color: "white",
    fontSize: "0.1em",
    textDecoration: "none",
    fontFamily: "monospace",
    fontWeight: "200"
});

The mixing works by overriding the default properties with the properties of the other styles consecutively. If one of the other styles' property is null, it is disregarded, the last property value will be used instead.

Basically it works like the rule of overriding. Property values will not be overriden until specified with a value that is NOT null.
