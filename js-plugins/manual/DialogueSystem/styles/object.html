Style object

If you want to temporarily change the text styles for one typing iteration, there are several ways to do it.

The easiest way though is by creating a new DialogueStyle object. Then calling the Type method with a DialogueData with it's style property being that style object. 

JavaScriptawait diaManager.Type("Hey BASIL, ", 1.4, new DialogueData({
    audio: "default",
    portrait: "kel_dark",
    delayAfter: 0.6,

    // increase font size ONLY in this call
    style: new DialogueStyle({ fontSize: "1.4em" })
}));

Style objects are also used by managers mainly to store it's text styles in it's style property.
