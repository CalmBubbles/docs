Adding styles

With CSS, applying styles to text has been easy. If you want apply styles and not affect previous text characters that have already been typed. Or if you want to do most things in the JavaScript side, you can.

In every manager, there's a style property. Which manages the styles of the text characters that are going to be typed. 

JavaScript// Unaffected by the change
await diaManager.Type("A simple use of styles is making text have a ", 1);

// The change
diaManager.style.color = "red";

// Affected by it
await diaManager.Type("red color.", 1);
await DialogueLoop.Delay(1.4);
await diaManager.ClearText();
await diaManager.Type("Even after clear, ", 1);
await DialogueLoop.Delay(0.5);
await diaManager.Type("the text is red.", 1);

This works similar to HTMLElement.style but with less stuff. You can check it's property list here.
