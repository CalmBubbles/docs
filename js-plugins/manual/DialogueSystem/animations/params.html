Animation parameters

Text animations can have extra properties to modify the animation's behavior, you can call them animation parameters. They are used to easily modify the variables of an animation during Type calls without changing the base/default properties.


Using parameters in idle

To use parameters in idle, in the Type call. Add a new Property object to the animationParams property of the DialogueData. 

JavaScript

Using parameters in transitions

In transitions, there is an animationParams property in DialogueAnimationPreset objects. Add a new Property object there. 

JavaScript

Making parameters

To make parameters, in the animation class, override or change it's OnSet method. Make it so, that the method takes in one parameter. This is where the passed values will go through. 

JavaScript

Now add the properties you want to your class. 

JavaScript

In the OnSet method, set the class properties by getting its values from the method parameter. The method parameter is a NamedArray, we can use its Find method and then use the Property's value property.

Now you can use the parameters.
