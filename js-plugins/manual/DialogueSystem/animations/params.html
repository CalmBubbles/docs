Animation parameters

Text animations can have extra properties to modify the animation's behavior, you can call them animation parameters. They are used to easily modify the variables of an animation during Type calls without changing the base/default properties.


Using parameters in idle

To use parameters in idle, in the Type call. Add a new Property object to the animationParams property of the DialogueData. 

JavaScriptawait diaManager.Type("Ueueuueueueueuueueu...", 1, new DialogueData({
    animation: "an animation name",
    
    // here
    animationParams : [
        new Property(
            "a property", // property name
            143 // property value
        )
    ]
}));

Using parameters in transitions

In transitions, there is an animationParams property in DialogueAnimationPreset objects. Add a new Property object there. 

JavaScriptdiaManager.animation.text.onAppend = new DialogueAnimationPreset({
    animation: "non existent animation, only for example",

    // here
    animationParams: [
        new Property(
            "another property", // name
            true // value
        )
    ],

    duration: 1,
    wait: false
});

Making parameters

To make parameters, in the animation class, override or change it's OnSet method. Make it so, that the method takes in one parameter. This is where the passed values will go through. 

JavaScriptclass AnimationName extends DialogueTextAnimation
{
    name = "example";

    OnSet (properties)
    {
        // ...
    }
}

Now add the properties you want to your class. 

JavaScriptclass AnimationName extends DialogueTextAnimation
{
    name = "example";

    aParameter = 0;
    anotherOne = false;
    oneMore = "";

    OnSet (properties)
    {
        // ...
    }
}

In the OnSet method, set the class properties by getting its values from the method parameter. The method parameter is a NamedArray, we can use its Find method and then use the Property's value property. 

JavaScriptclass AnimationName extends DialogueTextAnimation
{
    name = "example";

    // Don't set default values here
    // as they will be overridden
    //
    // I only added values to identify
    // the types I wan't them to be
    //
    //
    // And since they can't be modified
    // it is recommended to turn them private
    #aParameter = 0;
    #anotherOne = false;
    #oneMore = "";

    OnSet (properties)
    {
        // Place the default values here as `null` fallbacks
        this.#aParameter = properties.Find("aParameter").value ?? 10;
        this.#anotherOne = properties.Find("another parameter").value ?? true;
        this.#oneMore = properties.Find("one-more-param").value ?? "Just a string";
    }
}

Now you can use the parameters. 

JavaScriptawait diaManager.Type("I have no idea what text to put here.", 1, new DialogueData({
    animation: "example",
    animationParams : [
        new Property("aParameter", 8),
        new Property("one-more-param", false)
    ]
}));
