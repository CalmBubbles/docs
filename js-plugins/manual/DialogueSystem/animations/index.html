Animations

Animations can reduce boringness by a bunch. They can be played at all of the elements within the system. You can make transitions and cool idle animations and even express emotions.

Before learning about animations, remember that the animations within the system are split into 2 types. Keyframe animations and text animations.

Keyframe animations are based on CSS keyframe animations. All of the keyframe animations are used in transitions. These are easy to learn and is all tackled within the Basics.

Text animations on the other hand are quite complex. They are based around frame updates and code, and would require the knowledge of math and time within the system. This type of animation will take more than one manual section to tackle, but would be an awesome addition to dialogues.

This part of the manual includes

Basics The basics and how to use keyframe animations.
Text animations The simple parts of text animations and using them.
Animation parameters Working with text animation parameters.
Custom text animations Making custom text animations.
